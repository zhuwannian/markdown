# 加粗倾斜下划线等  markdown使用

## 加粗倾斜删除

### 代码（直接复制以下即可）：

```markdown

**粗体**

***加粗斜体***

*斜体*或_斜体_

~~删除线~~

```


### 显示效果：

**粗体**

***加粗斜体***

*斜体*或_斜体_

~~删除线~~


### 强调也可以直接插在文字中间：

### 代码（直接复制以下即可）：

```markdown

看我**变粗**了没有

un**frigging**believable

```

### 显示效果：

看我**变粗**了没有

un**frigging**believable
